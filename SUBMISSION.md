## 📝 Submission Descriptions

### Short Description (1-2 sentences)
> Flint is the open payment protocol for Solana with AI-powered escrow. Featuring QVAC integration for autonomous agent payments and Umbra SDK for private transactions.

### Long Description (for Colosseum)
> Flint solves the trust problem in AI commerce with a complete escrow framework. Sellers create payment requests, buyers fund escrow, and funds are released when work is delivered. AI-powered dispute resolution handles conflicts automatically.
>
> **Key Features**:
> - QVAC SDK integration for autonomous agent payments
> - Umbra SDK with ZK proofs for confidential transactions
> - Complete escrow flow: Accept → Fund → Deliver → Release/Dispute
> - Multi-token support: SOL, USDC, USDT
> - Account system with usernames and reputation badges
> - Dynamic fee structure: 1% (Free), 0.5% (Pro), 0.25% (Business)
> - Minimum invoice: $5, Minimum fee: $0.50
>
> **Tech Stack**:
> - Next.js 16 + TypeScript
> - Solana Web3.js
> - QVAC SDK (@qvac/sdk)
> - Umbra SDK (@umbra-privacy/sdk)
> - Supabase (free tier)
> - Vercel (free tier)
>
> **Why Solana**: Fast settlement (<1s), low fees, perfect for micropayments and AI agent commerce.
